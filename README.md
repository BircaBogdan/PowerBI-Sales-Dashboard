# Sales Overview Dashboard (Power BI)

## Overview

Interactive Sales Dashboard built in Power BI.

The dashboard provides insights into:

- Total Sales
- Sales in EUR
- Average Price
- Quantity Sold
- Monthly Sales Trend
- Top 5 Categories
- Top 8 Clients
- Sales by Status

- Fact
- Calendar
- DIM Category
- DIM Category Type
- DIM Clients
- DIM Status
- Plan
